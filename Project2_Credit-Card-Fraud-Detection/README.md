Credit Card Fraud Detection using Random Forest & SHAP
---------------------------------
🚀 Project Overview
------------

Developed a machine learning-based fraud detection system to identify suspicious credit card transactions using Random Forest, supported by explainable AI (SHAP) and an interactive dashboard.

This project was developed as part of a professional internship, where I contributed to building and evaluating the fraud detection model and developing analytical insights.

--------------------------------
💼 Business Problem
--------------------------------------
Credit card fraud leads to significant financial losses for banks and customers.

Early detection of fraudulent transactions is critical to:
- Prevent financial loss
- Protect customers
- Improve trust in financial systems

--------------------------------------------
🌍 Real-World Application Scenario
--------------------------------
This solution is designed for financial institutions to detect fraudulent transactions in real-time.

It demonstrates how machine learning models can be integrated into fraud monitoring systems to:
- Identify suspicious activity
- Reduce fraud risk
- Support fraud analysts

----
📊 Dataset Overview
----------------------
The project uses anonymized transaction data with highly imbalanced classes, representing real-world fraud scenarios.

-------------------
⚙️ Methodology
-----------------
1️⃣ Data Processing
- Handled class imbalance using weighted modeling
- Scaled transaction features

2️⃣ Modeling
- Random Forest Classifier
- Focused on recall to detect maximum fraud cases

3️⃣ Evaluation
- Precision, Recall, F1-score, ROC-AUC
- Emphasis on minimizing false negatives

-------------------
📈 Model Performance
------------------------------
The model achieved strong fraud detection performance with high recall, ensuring that most fraudulent transactions are correctly identified.

------------------------
📌 Key Insight
----------------------------
Handling class imbalance and optimizing recall is critical in fraud detection, as missing fraudulent transactions leads to higher financial risk.
